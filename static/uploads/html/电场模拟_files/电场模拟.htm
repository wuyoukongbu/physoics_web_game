如何使用

双击任意位置即可创建费用。

电荷下方的滑块可用于调整电荷的大小和符号。

您还可以通过单击并拖动电荷将其拖到屏幕上。

您可以右键单击费用以将其删除。您也可以单击费用并按下Delete键盘上的 键。

屏幕右侧的复选框可用于切换模拟的不同功能，显示有关电场的不同信息。

您可以通过点击屏幕上任意位置来勾选“测试电荷模式”复选框，并在屏幕上放置测试电荷。测试电荷会根据屏幕上电荷产生的电场移动。

在课堂上使用模拟

  • 将图像插入课堂幻灯片
  • 围绕直觉与现实为何不同的话题展开讨论
  • 工作表/引导问题

  • 创建费用配置
  • 模拟中的模型问题
